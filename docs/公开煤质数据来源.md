# 公开煤质数据来源说明

本项目的训练样本主要来自本地煤矿智能配煤系统数据库快照，同时补充少量公开煤质资料，以提高候选方案生成任务的数据真实性和煤质范围覆盖。

## 1. USGS COALQUAL 数据库

USGS COALQUAL 是美国地质调查局维护的煤质数据库。USGS 页面说明该数据库包含地理位置、现场观察、样品分析、煤层厚度、埋深、水分、灰分、硫分、热值以及主量/微量/痕量元素等信息。USGS 对 COALQUAL Version 3.0 的说明提到，该数据库源自 USCHEM，并包含超过 13,000 个样本的煤质数据。

用途：

- 作为论文中“真实公开煤质数据库”的来源说明；
- 后续如需扩大数据集，可从 COALQUAL Web Tool 导出样本 CSV 后，按 `data/raw/public_coal_quality_samples.csv` 的字段格式整理导入。

链接：

- https://www.usgs.gov/tools/coal-point-source-and-chemical-data-coalqual
- https://pubs.usgs.gov/publication/ds975

## 2. BMK 三层煤样工业分析数据

公开文献《Proximate Analysis, Sulfur, and Calorific Value in Determining Coal Quality》摘要给出了 BMK 三个煤层样本的总水分、灰分、挥发分、硫分、固定碳和发热量。本文将其中三条样本整理为调优项目的公开煤质样本。

用途：

- 构造低质煤限配样本；
- 构造低灰低硫高热值补偿样本；
- 扩展模型对极端灰分和高热值煤的识别能力。

链接：

- https://www.researchgate.net/publication/383971322_Proximate_Analysis_Sulfur_and_Calorific_Value_in_Determining_Coal_Quality

## 3. 准东五彩湾煤公开工业分析数据

公开论文和网页资料给出了准东五彩湾煤的工业分析/元素分析数据。例如：

- `M_ad=11.17%、A_ad=6.43%、V_ad=27.91%、S_t,ad=0.51%`；
- 另一组收到基数据包含 `M=25.01%、A=3.34%、V=31.12%、S=0.28%、Q_net.ad=20167.07 kJ/kg`，折合约 `4810 kcal/kg`。

用途：

- 构造低灰低硫但水分偏高的动力煤候选；
- 训练模型在水分约束严格时降低高水分煤比例；
- 支撑“真实煤质数据参与候选方案生成”的实验说明。

链接：

- https://eri.usst.edu.cn/html/2018/4/20180407.htm
- https://jns.usst.edu.cn/html/2019/1/20190105.htm

## 4. 使用方式

默认数据构建脚本会读取：

```bash
data/raw/public_coal_quality_samples.csv
```

并自动生成 `candidate_generation` 类型样本。执行：

```bash
python3 scripts/build_dataset.py --tasks all
```

如只生成候选方案任务样本：

```bash
python3 scripts/build_dataset.py --tasks candidate
```
